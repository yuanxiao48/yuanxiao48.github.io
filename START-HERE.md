# 从这里打开你的博客

双击这个文件：

```text
start-all.cmd
```

它会同时打开两个窗口：

- 博客预览：<http://127.0.0.1:4321/>
- 博客操作台：<http://127.0.0.1:4322/>

这两个黑色窗口要保持开着。关掉窗口之后，本地预览链接也会跟着失效；下次开机后再双击 `start-all.cmd` 就能重新打开。

## 操作台能改什么

- `站点`：网站名字、公开地址、主题色、默认深浅模式。
- `个人`：头像、名字、个人介绍、GitHub、邮箱。
- `首页`：首页大标题和轮换句子。
- `页面`：打开或隐藏友链、留言、番组、相册等页面。
- `板块`：上移、下移、开关侧边栏板块，也能把电脑端板块移到左侧或右侧。改完以后要点 `保存设置`。
- `音乐`：换网易云歌单、单曲、专辑，或者切成本地音乐。
- `评论`：开启评论区，并填写 Giscus / Twikoo / Waline 等评论服务配置。
- `预览`：在操作台里同时看电脑页面和手机页面。
- `文章`：快速创建新文章，也可以编辑已有文章，或把文章移到回收站。

## 文章怎么改

打开操作台的 `文章`：

1. 点某篇文章右侧的 `编辑`。
2. 在上面的表单里改标题、分类、标签、简介、正文。
3. 点 `保存文章`。

`移到回收站` 不会立刻永久删除，文件会放到项目里的 `.studio-trash/posts`。

## 板块怎么用

打开操作台的 `板块`：

- `电脑左侧` 和 `电脑右侧` 是电脑页面两边的侧边栏。
- `手机底部` 是手机页面底部出现的板块。
- `上移 / 下移` 改显示顺序。
- `开启 / 关闭` 控制这个板块是否显示。
- 电脑端每个板块还可以选 `左侧 / 右侧`，把它移动到另一边。
- 如果改乱了，点 `恢复默认板块`，再点 `保存设置`。

## 换成自己的网易云歌单

在操作台打开 `音乐`：

1. 平台选 `网易云`。
2. 类型选 `歌单`。
3. `ID 或关键词` 填你的网易云歌单 ID。
4. 保存设置。

歌单 ID 就是网易云歌单链接里 `id=` 后面的那串数字。

## 评论区

现在评论区入口已经打开，默认选的是 `Giscus`。Giscus 需要你自己的 GitHub 仓库开启 Discussions，然后在 <https://giscus.app/> 生成：

- Repo ID
- Category
- Category ID

把这些填进操作台的 `评论` 页面并保存，文章底部就会显示可用评论区。
